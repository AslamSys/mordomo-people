OPENCLAW AGENT

Sincronização de Dispositivo

O OpenClaw usa a biblioteca Baileys (embebida no NodeJS) para gerenciar o WhatsApp de forma nativa. Siga os passos para ativar a conexão socket.

1
Acesse via SSH o núcleo do sistema Entre no terminal do servidor Orange Pi usando o usuário mordomo.
2
Acesse os logs do Agente Node.js Execute o comando de monitoramento contínuo: docker logs -f mordomo-openclaw-agent
3
Escaneie o QR Code em ASCII O OpenClaw gerará um bloco enorme de caracteres formando um código QR na sua tela preta. Leia com seu WhatsApp para gerar a sessão (Ela ficará armazenada no volume persistente).
⚠️
Limitação Arquitetural de Segurança: Para evitar exportação da string de sessão pela rede, o OpenClaw não expõe endpoints legíveis de sua API Baileys para este Dashboard em Python. O pareamento por Pairing Code ou leitura visual deve ser feito em ambiente restrito originado da máquina host.
